# Learning Plan

This project is structured into phases that progressively build knowledge of system administration and IT support.

---

## Phase 0 — Planning and Documentation

Goals:

- Define project scope
- Create documentation structure
- Plan lab architecture

Skills practiced:

- technical planning
- documentation practices
- project organization

---

## Phase 1 — Virtualization Basics

Topics:

- virtual machines
- resource allocation
- virtual networking
- snapshots

Skills practiced:

- creating and managing VMs
- understanding virtualization concepts
- safe experimentation

---

## Phase 2 — Windows Server Setup

Topics:

- installing Windows Server
- server configuration
- system settings

Skills practiced:

- Windows administration
- system configuration
- server management

---

## Phase 3 — Active Directory

Topics:

- domain controller setup
- user management
- domain authentication

Skills practiced:

- Active Directory basics
- account management
- authentication workflows

---

## Phase 4 — Client Systems

Topics:

- Windows client setup
- domain joining
- login management

Skills practiced:

- workstation support
- user login troubleshooting

---

## Phase 5 — Networking Services

Topics:

- DNS
- DHCP
- IP addressing

Skills practiced:

- network configuration
- troubleshooting connectivity issues

---

## Phase 6 — Linux and SSH

Topics:

- Linux server setup
- SSH access

Skills practiced:

- remote administration
- command line basics

---

## Phase 7 — Help Desk Scenarios

Topics:

- simulated user problems
- troubleshooting workflows
- documentation

Skills practiced:

- root cause analysis
- systematic troubleshooting
- technical communication
